# Express Server

A simple Express.js server for the Jobify application.

## Project setup

```bash
$ pnpm install
```

## Run the server

```bash
# development with auto-reload
$ pnpm run dev

# production mode
$ pnpm run start
```

## API Endpoints

- `GET /api/health` - Health check

## Environment Variables

Create a `.env` file in the server directory:

```
PORT=4000
```
